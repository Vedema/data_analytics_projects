# Source Code

This directory contains Python modules and scripts for the data analytics projects.

## Guidelines

- Organize code into logical modules
- Write reusable functions
- Include docstrings for all functions and classes
- Follow PEP 8 style guidelines

## Typical Files

- `data_processing.py`: Functions for data cleaning and transformation
- `visualization.py`: Custom plotting functions
- `analysis.py`: Statistical analysis functions
- `utils.py`: Utility functions used across projects
